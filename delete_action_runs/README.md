## How to run

**raw download**

```sh
mkdir -p foo && cd foo
curl https://gist.githubusercontent.com/guitarrapc/26cf681731965acc12c3cc6b3a867204/raw/indes.js > index.js
npm install @octokit/rest --save
```

Run Debug on VSCode as NodeJS.

**clone**

```sh
npm ci
```

Run Debug on VSCode as NodeJS.
