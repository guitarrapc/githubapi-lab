# README


## How to run

**raw download**

```sh
mkdir -p foo && cd foo
curl -sSf -L https://raw.githubusercontent.com/guitarrapc/githubapi-lab/main/delete_artifacts/index.js > index.js
npm install @octokit/rest --save
code .
```

Run Debug on VSCode as NodeJS.

**clone**

```sh
cd delete_artifacts
npm ci
code .
```

Run Debug on VSCode as NodeJS.
